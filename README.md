- Node.js (v16 or higher)

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd Kerenmehas
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

The site will be available at `http://localhost:3000`
